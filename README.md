# 💰 Credit Wise: Loan Approval Prediction System

A Machine Learning-based classification project designed to predict whether a loan application should be approved or rejected based on applicant details. 🤖

## 📋 Project Overview
The loan approval process is often time-consuming and manual. This project automates the decision-making process using the **K-Nearest Neighbors (KNN)** algorithm. By analyzing factors like income, credit history, and education, the model provides an instant prediction, helping financial institutions streamline their workflow.

## 🛠️ Tech Stack
* **Language:** Python 🐍
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib
* **Algorithm:** K-Nearest Neighbors (KNN) 🤖
* **Environment:** VS Code / Jupyter Notebook

## 📊 Dataset Features
The model processes the following applicant information:
* **Gender, Marital Status, and Dependents**
* **Education & Employment Status**
* **Applicant & Co-applicant Income**
* **Loan Amount & Loan Amount Term**
* **Credit History** (Key predictor for approval)

## 🚀 Key Features
* **Data Cleaning:** Handled missing values (NaN) to ensure data quality.
* **Label Encoding:** Converted categorical data into numerical format (0s and 1s) for model compatibility.
* **Visualization:** Used Matplotlib to understand data distributions.

## 📈 Future Enhancements
* **Streamlit Integration:** Building a web interface for real-time user input.
* **Model Optimization:** Implementing Feature Scaling and Hyperparameter Tuning to improve accuracy.
* **Comparative Analysis:** Comparing KNN results with Random Forest and Logistic Regression.
